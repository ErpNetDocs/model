Attributes

Name Type Description
IsActive boolean Indicates whether the web site is active and will be instantiated upon next web server restart.
IsPrivate boolean Specifies that the web site address will not be publicly listed. The web site itself will still be publicly accessible; only its URL would not be listed in the auto-discovery service.
Notes string (max) nullable Notes for this WebSite.
RelativeUrl string (254) nullable The relative Url of the site. This is the text after the first slash after the protocol and host name. The text should not include the protocol and host name. NULL means that the site will be hosted as the root site in the speicified web host.
SettingsJson string (max) nullable The field specifies the JSON settings for this website. NULL means that there are no specific settings for this website.
Uin string (64) nullable For multi-instance databases, specifies the instance on which to start the web site. The instance is specified by its UIN (Unique Instance Name). When NULL (which is the default and suggested value), the site will be started on the primary instance.
WebSiteType WebSiteType The type of web site - Api, Client Center, Id, etc.

References

WebHost WebHosts (nullable) The web host in which to host the site.
