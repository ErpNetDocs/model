    Logistics.Wms.WarehouseRequisitionExecutions Entity

    Execution ledger for warehouse requisitions. Entity: Wms_Warehouse_Requisition_Executions (Introduced in version 20.1)

    Attributes

    Name Type Description
    CreationTimeUtc datetime The exact time in UTC, when the execution was created in the system. [Required]
    ExecutedQuantity decimal The executed quantity (in the measurement unit of the requisition line). [Required]
    ExecutionType string Type of execution: P=Plan created; C=Requisition completed. [Required]
    Id guid
    IsFinal boolean Specifies whether this execution will finalize the execution of the requisition line, regardless of any remaining quantities. [Required]
