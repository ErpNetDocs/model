Parameters

  • accessKeyId
    The access key id
    Type: guid

  • userToken
    A proof token, identifying the logged in user. E.g. It could be an id token, or an access token. Required when the currently logged user is different from the public user.
    Type: string
    Optional: True
    Default Value: null
