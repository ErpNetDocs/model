    R33201 WarehousePolicies - Zone Policy

    Name Value
    Code R33201
    Entity Logistics.Wms.WarehousePolicies Entity
    Name Zone Policy
    Attribute -
    Layer Back-End
    Events ClientCommit
    Priority Normal
    Modify NO
    Applicable Legislations ALL // no condition needed
    Action Validate that:
    If (PolicyKind LIKE 'Zone%')
    , then:
    - Zone != null
    - ProductType == null
    - ProductGroup == null
    - Product == null
    Description Policies whose kind starts with 'Zone', such as 'Zone Type', aplly to a specific zone.
    For this reason, the rule makes sure that:
    - there is a Zone specified in the policy
    and
    - the policy applies to the entire zone, therefore no other properties, such as Product, Product Type or Product group are set.
    Message Invalid information specified for '{PolicyKind}' policy kind.
    Policies whose kind starts with 'Zone' apply to a specific zone.
    Please make sure that the 'Zone' field has a value and the 'Product', 'Product Group' and 'Product Type' fields are blank.
