R38826 Activity - Set Responsible Party From Assigned To User

Code R38826
Entity General.Activities.Activities Entity
Name SetResponsiblePartyFromAssignedToUser
Attribute ResponsibleParty
Layer Front-End
Events AttributeChanged(AssignedToUser)
Priority Normal
Modify YES
Action Set value:

ResponsibleParty = AssignedToUser.Person
Description To maintain consistency between assigned users and responsible parties, this rule ensures that, upon modification of the Assigned to User field, the Responsible Party field is synchronized with the Person associated with the new User. If no Person is linked, the field remains empty.
Message -
Version Introduced: v.25
Revocable YES
