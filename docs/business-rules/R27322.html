    R27322 StoreTransaction - Parent Store Order Line

    Code R27322
    Entity Logistics.Inventory.StoreTransactions Entity
    Name ParentStoreOrderLine
    Attribute ParentStoreOrderLine
    Layer Back-End
    Events Releasing
    Modify NO
    Action For every line:

    If "StoreTransaction.ParentStoreOrder" has value, validate that "StoreTransactionLine.ParentStoreOrderLine" has value.
    Description If the field "Parent Store Order" has a value, then the current Store Transaction refers a particular Store Order. This means that the field "Parent Store Order Line" has to be filled in all document lines.
    Message The field 'Parent Store Order Line' in line '{StoreTransaction.LineNo}' is empty. The field must have a value in all lines because the field 'Parent Store Order' in the 'Transaction' panel refers to a specific document.



    Introduced In Version 2017
    Revocable NO
