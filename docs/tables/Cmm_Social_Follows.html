    Table Cmm_Social_Follows

    Entity

    Entity: Communities.Social.Follows

    Denotes a user following an object in the system. The user will be notified for events about the object. Entity: Cmm_Social_Follows (Introduced in version 22.1.6.73)

    Summary

    Name Type Description
    Creation_Time_Utc datetime The exact server time (in UTC), when the follow was created.
    Data_Object_Id uniqueidentifier The object being followed by the user.
    Row_Version timestamp
    Social_Follow_Id uniqueidentifier PK
    User_Id uniqueidentifier The user which follows the object.
