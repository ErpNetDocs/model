    Table Sys_Resolved_Conflicts

    Entity

    Entity: Systems.Monitoring.ResolvedConflicts

    Contains records of conflicts, which were automatically resolved by update procedures. Entity: Sys_Resolved_Conflicts

    Summary

    Name Type Description
    Conflict_Description nvarchar(400) ML, Readonly Description of the conflict.
    Resolve_Confirmed_By_User bit True, when the conflict resolution was manually confirmed by user.
    Resolve_Confirmed_Time datetime Readonly Time when the conflict resolution was confirmed by the user.
    Resolve_Description nvarchar(400) ML, Readonly Description of the resolution of the conflict.
    Resolved_Conflict_Id uniqueidentifier PK Unique identification of the resolved conflict.
    Resolved_Time datetime Readonly Time when the resolution of the conflict was made.
    Revised_By_User bit Readonly True, when the conflict resolution was revised (reviewed) manually by user.
    Row_Version timestamp
    Table_Name nvarchar(64) ML, Readonly Name of the table in which the conflict has occurred.
    URL nvarchar(254) Readonly URL of the item (the row) for which the conflict occurred.
