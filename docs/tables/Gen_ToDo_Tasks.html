    Table Gen_Todo_Tasks

    Entity

    Entity: Projects.Todo.Tasks

    Represents a task, such as a piece of work or personal item, that can be tracked and completed. Entity: Gen_Todo_Tasks (Introduced in version 23.1.1.39)

    Summary

    Name Type Description
    Assigned_To_User_Id uniqueidentifier The user, to whom the todo is assigned.
    Completed_Date_Time_Utc datetime Readonly Indicates (in UTC) when the task was completed.
    Due_Date date Indicates when the task should be finished.
    Importance nvarchar(1) Allowed: L, N, H The importance of the task.
    Notes nvarchar(max)
    Owner_User_Id uniqueidentifier The user, who created the todo and owns it.
    Remind_Time_Utc datetime When to remind the assigned user for the task (in UTC).
    Row_Version timestamp
    Social_Group_Id uniqueidentifier When not null, indicates that the todo is contained in and managed by the specified social group.
    State nvarchar(1) Allowed: N, P, W, C Indicates the current task state.
    Title nvarchar(254) A brief description of the task.
    Todo_Task_Id uniqueidentifier PK
