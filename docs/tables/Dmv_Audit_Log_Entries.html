    View Dmv_Audit_Log_Entries

    Entity

    Entity: Systems.Monitoring.AuditLogSummary

    Information about the count and the size of audit log entries, grouped by application name, event class, entity name and event time. Entity: Dmv_Audit_Log_Entries (Introduced in version 24.1.3.48)

    Summary

    Name Type Description
    Application_Name nvarchar(64) The client application that triggered the events. Null when unknown or N/A.
    Entity_Name nvarchar(64) The entity, which is being referenced by the events. Null when unknown or N/A.
    Entries_Count int Total number of audit log entries.
    Event_Class nvarchar(1) Allowed: E, A, S The event primary classification, which shows the source of the event. E=Entity methods; A=Auth events; S=Server events.
    Total_Size_MB decimal(12, 3) Total size of the audit log entries in megabytes.
    Year nvarchar(30) Year when the events occurred.
