    Table Gen_Document_Types

    Entity

    Entity: Systems.Documents.DocumentTypes

    List of user-defined document types. Each type has associated system entity (object class). Entity: Gen_Document_Types

    Summary

    Name Type Description
    Access_Key_Id uniqueidentifier
    Code nvarchar(16) Unique descriptive code of the document type
    Create_Fulfillments_On_Completion bit When document state is changed to Completed, creates completed document fulfillments for the parent document. Used only for logistic documents.
    Create_Manully bit 0 if documents with this document type only can be generated; 1 - the user can create documents with this type
    Description nvarchar(max) ML
    Disallow_Opposite_Values_Generation bit Disallow the generation of decreasing scalar values (values with opposite directions than the original values determined by the parent document) through this document type
    Entity_Name nvarchar(64) System entity, which contains the documents
    Generate_Single_Document bit Create maximum one document when documents from this type are generated by parent documents (usually from another document type)
    Id uniqueidentifier PK The unique Id of the document type
    Notes nvarchar(254)
    Row_Version timestamp
    Schema_XML nvarchar(max) Obsolete. Not used.
    Sequence_Id uniqueidentifier The sequence that will be used to give new numbers to the documents of this type
    Track_Attribute_Changes nvarchar(3) Allowed: DEF, ENA, DIS Enable/disable attributes change tracking for documents from this type. "Default" means that changes will be tracked for all documents, except transit and adjustment documents.
    Track_Print_Images nvarchar(3) Allowed: DNT, SDC Enable/disable print images tracking for documents from this type.
    Transitional_Document bit If checked determines that the documents from this type are automatically managed by the system and don't require management from the users
    Type_Name nvarchar(254) ML Description of the document type
