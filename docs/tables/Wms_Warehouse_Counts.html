Table Wms_Warehouse_Counts

Entity

Entity: Logistics.Wms.WarehouseCounts

Captures raw physical count entries recorded in the warehouse operations before they are processed or consolidated by the system. Entity: Wms_Warehouse_Counts (Introduced in version 26.2.1.4)

Summary

Name Type Description
Creation_Time_UTC datetime Readonly The timestamp when the counting entry was recorded (UTC).
Creation_User_Id uniqueidentifier Readonly The user who recorded the counting entry.
Logistic_Unit_Id uniqueidentifier Readonly The logistic unit involved in the counting, when applicable.
Lot_Id uniqueidentifier Readonly Batch/lot of the product, when applicable.
Product_Id uniqueidentifier Readonly The product that was counted.
Quantity decimal(12, 3) Readonly The quantity entered during the counting process, in the entered measurement unit.
Quantity_Base decimal(12, 3) Readonly The converted quantity in the product’s base measurement unit.
Quantity_Unit_Id uniqueidentifier Readonly The measurement unit used when entering the quantity.
Row_Version timestamp
Serial_Number_Id uniqueidentifier Readonly Serial number, when serialized tracking is enabled.
Session int Readonly The counting session number to which this entry belongs.
Source_Document_Id uniqueidentifier Readonly Requisition, reconciliation, or other document which initiated the counting.
Standard_Quantity_Base decimal(12, 3) Readonly The theoretical quantity in the base measurement unit, calculated according to the product’s standard measurement ratio.
Variant_Id uniqueidentifier Readonly Product variant (e.g. size, color, configuration), when tracked.
Warehouse_Count_Id uniqueidentifier PK
Warehouse_Location_Id uniqueidentifier Readonly The warehouse location where the counting occurred.
Warehouse_Order_Id uniqueidentifier Readonly Warehouse order which initiated the counting.
