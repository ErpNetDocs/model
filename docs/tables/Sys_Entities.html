    Table Sys_Entities

    Entity

    Entity: Systems.Security.EntitySettings

    Contains entities, which have secured access. Entity: Sys_Entities

    Summary

    Name Type Description
    Access_Key_Id uniqueidentifier The access key, required to access the secured entity.
    Display_Text_Format nvarchar(128) Interpolated string, containing the default format for displaying values of the entity. NULL means to use the system-wide default.
    Entity_Id uniqueidentifier PK
    Entity_Name nvarchar(64) The system name of the entity, which is being secured.
    Log_Create bit Specifies whether to log every insert for this entity.
    Log_Delete bit Specifies whether to log every delete for this entity.
    Log_Read_By_Id bit Specifies whether to log every load by Id for this entity.
    Log_Read_Many bit Specifies whether to log every load of many records for this entity.
    Log_Update bit Specifies whether to log every update for this entity.
    Row_Version timestamp
    Track_Changes_Level tinyint Allowed: 0, 1, 2, 3, 4 The track changes level for the entity
