    Summary

    Name Type Description
    Line_No int
    Lot_Id uniqueidentifier When not null, indicates a specific lot should be purchased.
    Notes nvarchar(254)
    Product_Description nvarchar(254) ML The description of the required product. When Product is set, this is copied initially from the product name. When Product is null, this contains the manually entered description of the desired product.
    Product_Id uniqueidentifier The required product. When null, the product is unknown to the requisitor and only a description is supplied to the purchase department.
    Quantity decimal(18, 3) The required quantity of the product.
    Quantity_Base decimal(18, 3) Readonly The equivalence of Quantity in the base measurement category of the product.
    Quantity_Unit_Id uniqueidentifier The measurement unit of Quantity.
    Required_Delivery_Date datetime The desired delivery date. Initially set to the required delivery date in the requisition header or if it is empty - to the document date plus the products lead time.
    Requisition_Id uniqueidentifier
    Requisition_Line_Id uniqueidentifier PK
    Row_Version timestamp
    Standard_Quantity_Base decimal(18, 3) Readonly The theoretical quantity in base measurement unit according to the current measurement dimensions for the product. Used to measure the execution.
    Suggested_Supplier_Id uniqueidentifier When the requisitor knows the supplier or has a supplier preference it is denoted in this field.
