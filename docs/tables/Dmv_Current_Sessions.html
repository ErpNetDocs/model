    View Dmv_Current_Sessions

    Entity

    Entity: Systems.Monitoring.CurrentSessions

    Sessions dynamic management view. Entity: Dmv_Current_Sessions (Introduced in version 23.1.1.52)

    Summary

    Name Type Description
    Absolute_Expiration_Time datetime Absolute expiration time of the session. Not empty if the session is created by a service appllication.
    Applications nvarchar(64) A comma separated list of client applications that share this session.
    Current_Requests_Count int The requests count in the time of the request.
    Device nvarchar(64) The name of the user's device.
    Download_MB decimal(12, 3) The downloaded megabytes at the time of the request.
    Last_Request_Time datetime The last request time.
    Start_Time datetime The login time of the session.
    Total_Requests_Count bigint The total request count at the time of the request.
    Upload_MB decimal(12, 3) The uploaded megabytes at the time of the request.
    User nvarchar(64) The login name of the session's user.
