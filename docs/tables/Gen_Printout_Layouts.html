    Table Gen_Printout_Layouts

    Entity

    Entity: Systems.Documents.PrintoutLayouts

    Contains design layouts for document printouts. Entity: Gen_Printout_Layouts

    Summary

    Name Type Description
    Binary_Layout varbinary The printout layout, when the format requires binary storage. Alternative to Layout
    Data_Source_Id uniqueidentifier The data source for the printout
    Document_Entity_Name nvarchar(64) The entity name of the document type e.g. Crm_Sales_Orders, Inv_Store_Orders etc.
    Layout nvarchar(max) The textual representation of the printout layout, when the format requires text representation. Alternative to Binary_Layout
    Layout_Format nvarchar(32) Format specifier of the layout. Recognized by the application
    Notes nvarchar(254)
