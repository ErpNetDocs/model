    Table Log_Carriers

    Entity

    Entity: Logistics.Shipment.Carriers

    Represents a carrier agreement. One actual carrier can have different agreements over time or with different enterprise companies. Entity: Log_Carriers

    Summary

    Name Type Description
    Agreement_End_Date date The ending date of the agreement with this carrier. NULL for agreements which are still active.
    Agreement_Start_Date date The start date of the agreement with this carrier.
    Carrier_Id uniqueidentifier PK
    Code nvarchar(16) Unique carrier code (or call sign). The code is unique within all enterprise companies.
    Enterprise_Company_Id uniqueidentifier The enterprise company which signed the carrier agreement.
    Is_Active bit Specifies whether the carrier agreement is active.
    Notes nvarchar(max)
    Row_Version timestamp
    Supplier_Id uniqueidentifier The supplier contract with the carrier.
