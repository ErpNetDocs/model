    Table Pos_Operators

    Entity

    Entity: Crm.Pos.Operators

    Represents one operator (person) in one POS location. Entity: Pos_Operators (Introduced in version 19.1)

    Summary

    Name Type Description
    Default_Pos_Terminal_Id uniqueidentifier The default POS terminal for this opertor. NULL when there is no default.
    Is_Active bit Indicates whether this operator is active and can be chosen for new records.
    Pos_Location_Id uniqueidentifier The POS location where this operator works.
    Pos_Operator_Code nvarchar(16) Operator code. Unique within the Pos Location.
    Pos_Operator_Id uniqueidentifier PK
    Pos_Role_Id uniqueidentifier The role, assigned to the operator. The role indicates the permissions of the operator for this POS location.
    Row_Version timestamp
    Starting_Date date The first date, when the operator has started working for this POS location.
    Termination_Date date The date, when the operator has ceased working in this POS location. NULL means, that the operator is still working or the termination date is still unknown.
    User_Id uniqueidentifier The login user for this POS operator.
