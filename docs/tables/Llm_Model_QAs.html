    Table Llm_Model_QAs

    Entity

    Entity: Projects.AI.ModelQAs

    Questions and desired answers, forming the actual knowledge base of the model. Entity: Llm_Model_QAs (Introduced in version 24.1.1.93)

    Owner Tables Hierarchy

    • Llm_Models

    Summary

    Name Type Description
    Added_By_User_Id uniqueidentifier Readonly The user which added the QA.
    Answer nvarchar(max) Desired answer.
    Creation_Time_Utc datetime Readonly The time when the QA was created.
    Deactivaton_Time_Utc datetime Readonly The time when the QA was deactivated.
    Is_Active bit Indicates whether to include this QA in future builds.
    Model_Id uniqueidentifier The model to which the question belongs.
    Model_QA_Id uniqueidentifier PK
    Notes nvarchar(max)
    QA_Type nvarchar(1) Allowed: T, V Specifies whether the QA is training or validation. Validation QAs are used to test the correctness of the model and are only reported in the build logs.
    Question nvarchar(max) User question.
    Row_Version timestamp
    Section nvarchar(128) Free text organizational section of the QA.
