    Table Sys_Information_Messages

    Entity

    Entity: Systems.Monitoring.InformationMessages

    Contains information messages, generated by various system processes. Entity: Sys_Information_Messages

    Summary

    Name Type Description
    Information_Message_Id uniqueidentifier PK
    Information_Message_Time datetime Readonly Date and time when thet message occurred.
    Information_Message_Type nvarchar(3) Allowed: INF, WRN, ERR, Readonly Type of the information message: 'INF' = Information, 'WRN' = Warning, Error = 'ERR'
    Machine_Name nvarchar(128) Readonly Machine name of the computer from which the process which creates the message has been initiated.
    Message nvarchar(max) Readonly The actual content of the information message.
    Process_Description nvarchar(254) Readonly Name or description of the process that creates the message.
    Row_Version timestamp
    URL nvarchar(254) URL of the record that has posted the information message
    User_Name nvarchar(128) Readonly Login name of the user that has initiated the process which creates the message.
