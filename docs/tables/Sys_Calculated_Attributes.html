    Table Sys_Calculated_Attributes

    Entity

    Entity: Systems.Bpm.CalculatedAttributes

    User-defined read-only calculated attribute. Entity: Sys_Calculated_Attributes

    Summary

    Name Type Description
    Calculated_Attribute_Id uniqueidentifier PK
    Caption nvarchar(512) ML The multi-language caption, used to display the attribute.
    Hint nvarchar(max) ML The hint, which is displayed alongside the attribute.
    Is_Active bit True if the attribute is activated and added to the repository.
    Name nvarchar(128) The unique name of the attribute within the repository.
    Notes nvarchar(max)
    Repository_Name nvarchar(128) The repository, for which the attribute is defined.
    Row_Version timestamp
    Starting_Expression_No int The expression, from which the calculation starts. The result of the expression gives the value of the calculated attribute.
