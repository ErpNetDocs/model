    Table Sys_Data_Sources

    Entity

    Entity: Systems.Documents.DataSources

    Contains user-defined data sources, which retrieve rows from multiple queries. Entity: Sys_Data_Sources

    Summary

    Name Type Description
    Base_Query_Name nvarchar(128) The name of the query or table that is used for root reference point of the loaded data.
    Data_Source_Id uniqueidentifier PK
    Data_Source_Name nvarchar(64) The name of the data source.
    Data_Source_Type nvarchar(1) Allowed: M, D, S 'M' = MULTI-TABLE (many tables); 'D' = MASTER-DETAIL (two tables); 'S' = SINGLE-TABLE
    Row_Version timestamp
    Show_Parent_Tables bit Indicates whether the parent nodes in the Reference_Path in Sys_Data_Source_Queries_Table are automaticaly included in the report or not.
