    Table Sys_Reports

    Entity

    Entity: Systems.Core.Reports

    User-defined reports. Reports retrieve data from data sources and present it in a user-defined layout. Entity: Sys_Reports

    Summary

    Name Type Description
    Data_Source_Id uniqueidentifier Report data source. If specified should be based on the query in Query_Name. NULL means that the report is based directly on the query
    Layout_Format nvarchar(32) Format specifier of the layout. Recognized by the application
    Notes nvarchar(512)
    One_Row bit True means that report can be used for one-row forms only. False means that the report can be used for navigators only.
    Query_Name nvarchar(128) The query on which the report is based
    Report_Binary_Layout varbinary The printout layout, when the format requires binary storage. Alternative to Report_Layout
    Report_Id uniqueidentifier PK
    Report_Layout nvarchar(max) The textual representation of the printout layout, when the format requires text representation. Alternative to Report_Binary_Layout
    Report_Name nvarchar(254) Multi-language report name
    Row_Version timestamp
    Template_Filter_Xml nvarchar(max) Specifies the filter condition, which must be satisfied in order for this report to be prioritized for printing as a template sub-report. NULL when the current report cannot be used as template sub-report.
    Template_Priority int Specifies the priority of the current report, relative to other reports. Used only when choosing which report to show as a template subreport, NULL otherwise.
