Table Eam_Managed_Asset_Links

Entity

Entity: Applications.AssetManagement.ManagedAssetLinks

Used to associate the Managed Asset with a set of assets comprising of a Fixed Asset, a Rental Asset, a Service Object and a Vehicle. Indicates the currently managed set and dates of management. Entity: Eam_Managed_Asset_Links (Introduced in version 25.1.2.68)

Owner Tables Hierarchy

  • Eam_Managed_Assets

Summary

Name Type Description

Fixed_Asset_Id uniqueidentifier The linked fixed asset that is managed. NULL means that no fixed asset is linked.
From_Date datetime Start date of asset group management.
Is_Current bit Indicates that this group of assets is currently being managed.
