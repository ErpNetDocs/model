    Table Srv_Service_Activity_Materials

    Entity

    Entity: Applications.Service.ServiceActivityMaterials

    Contains the materials, which were actually used during the service activity (repair). Entity: Srv_Service_Activity_Materials

    Owner Tables Hierarchy

    • Srv_Service_Activities
    • Cm_Activities
    • Gen_Documents

    Summary

    Name Type Description
    Covered_By_Guarantee bit True when the used material is covered by the guarantee.
    Line_No int Consecutive line number, unique within the document. Usually is increasing in steps of 10, like in 10, 20, 30, etc.
    Line_Store_Id uniqueidentifier The store from which the product was taken. NULL = use the store from the header
    Product_Id uniqueidentifier The product, which was used as material.
    Quantity decimal(18, 3) Quantity of the product, that was used
    Quantity_Base decimal(18, 3) Readonly The equivalence of Quantity in the base measurement category of the product.
    Quantity_Unit_Id uniqueidentifier The measurement unit of Quantity. Initially is set to the default unit for the product
    Row_Version timestamp
    Service_Activity_Id uniqueidentifier
    Service_Activity_Material_Id uniqueidentifier PK
    Service_Object_Id uniqueidentifier The service object for which the material is used. NULL means unkown object or N/A
    Standard_Quantity_Base decimal(18, 3) Readonly The theoretical quantity in base measurement unit according to the current measurement dimensions for the product. Used to measure the execution.
