    Table Sys_User_Business_Rules

    Entity

    Entity: Systems.Bpm.UserBusinessRules

    Represents user-defined business rule. Entity: Sys_User_Business_Rules

    Summary

    Name Type Description
    Icon varbinary Visual icon of the rule in .PNG format.
    Is_Active bit Specifies whether the rule is activated.
    Layer nvarchar(3) Allowed: FTE, BKE, COM Specifies in which layers the rule will be available. The available events and actions depend on the chosen layer. Allowed values: FTE=Front-End, BKE=BackEnd, COM=Common (both).
    Notes nvarchar(max)
    Repository_Name nvarchar(128) The name of the repository, for which this business rule is defined.
    Row_Version timestamp
    Script_Language nvarchar(16) Allowed: Integrated, CSharp The programming language used to define the rule actions.
    Script_Text nvarchar(max) The program code used to define the rule actions.
    User_Business_Rule_Code nvarchar(16)
    User_Business_Rule_Id uniqueidentifier PK
    User_Business_Rule_Name nvarchar(254) ML
    User_Startable bit Specifies, that the rule can be manually started by the user.
