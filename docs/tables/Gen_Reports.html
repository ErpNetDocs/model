    Table Gen_Reports

    Note

    OBSOLETE! Do not use!

    Entity

    Entity: General.Reports

    Contains user-defined data-sources. The data sources is a user-defined way (similar to query designer) to extract, filter, join and transform data for presentation purposes. Entity: Gen_Reports (Obsoleted in version 22.1.6.73)

    Summary

    Name Type Description
    Is_Table bit If True the Query_Name is name of a table, otherwise is name of a query that can be filtered with Filter_Xml.
    Query_Name nvarchar(128) The name of the query or table that is used for root reference point of the loaded data.
    Report_Id uniqueidentifier PK
    Report_Name nvarchar(64)
    Row_Version timestamp
    Show_Parent_Tables bit Indicates whether the parent nodes in the Reference_Path in Sys_Data_Source_Queries_Table are automaticaly included in the report or not.
