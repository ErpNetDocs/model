Table Sec_Access_Tokens

Entity

Entity: Systems.Security.AccessTokens

It keeps track of "grants" issued by Identity Server - Personal Access Tokens (PATs), refresh tokens, device codes, and consents. Entity: Sec_Access_Tokens (Introduced in version 26.2.0.13)

Summary

Name Type Description
Access_Token_Id uniqueidentifier PK
Access_Token_Key nvarchar(256) The unique key of the token
Actor nvarchar(128) For whom the token was issued (might be user email)
Client nvarchar(128) The client application
Consumed_Timestamp datetime When it was used up (for one-time grants).
Creation_Timestamp datetime When the token was created
Data nvarchar(max) Token contents
Description nvarchar(128)
Expire_Timestamp datetime When the token expires
Row_Version timestamp
Session nvarchar(128) Session parameter
Trusted_Application_Id uniqueidentifier Sec_Trusted_Applications // Matched trusted application (from Client)
Type nvarchar(64) Type of the token - reference_token, refresh_token, etc.
User_Id uniqueidentifier Matched user (from the actor)
