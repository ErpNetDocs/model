    View Dmv_License_History

    Entity

    Entity: Systems.Monitoring.LicenseHistory

    Historical licensing events for the instance. Entity: Dmv_License_History (Introduced in version 24.1.3.88)

    Summary

    Name Type Description
    License_Code nvarchar(64) The code of the module or functionality, which is licensed.
    License_Count int The number of licenses given (+) or taken (-).
    License_Description nvarchar(max) Description of the license code.
    License_End_Date date The date (inclusive), until the license is active.
    License_Scale_Type nvarchar(64) Denotes whether the license supports counts, different from 0 and 1.
    License_Start_Date date The date (inclusive), from which the license is active.
    Licensing_Date date The date, when the license was issued.
    Licensing_Event nvarchar(128) Unique licensing event key. Can be used to group all license codes given with a single licensing event.
