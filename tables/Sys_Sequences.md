# Table Sys_Sequences


## Entity

Entity: [Systems.Core.Sequences](~/entities/Systems.Core.Sequences.md)

Stores sequence definitions used to generate and control incremental identifiers for different entity types. Entity: Sys_Sequences (Introduced in version 26.2.1.31)

## Summary

| Name | Type | Description |
| - | - | --- |
|[Entity_Type](#entity_type)|`nvarchar(64)` |Specifies the entity type to which the sequence applies.|
|[Filter_Condition_Xml](#filter_condition_xml)|`nvarchar(max)` |Stores optional XML filter criteria used to select the appropriate sequence definition.|
|[Max_No](#max_no)|`nvarchar(16)` |Defines the maximum allowed value for the sequence.|
|[Next_No](#next_no)|`nvarchar(16)` |Holds the next value to be assigned by the sequence.|
|[Row_Version](#row_version)|`timestamp` |Used for concurrency control to track row changes.|
|[Sequence_Id](#sequence_id)|`uniqueidentifier` `PK`|Unique identifier of the sequence record.|
