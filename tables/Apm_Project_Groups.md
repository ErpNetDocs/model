# Table Apm_Project_Groups


## Entity

Entity: [Projects.Agile.ProjectGroups](~/entities/Projects.Agile.ProjectGroups.md)

Logical grouping of projects for shared configuration, such as project areas, milestones, etc. Entity: Apm_Project_Groups (Introduced in version 26.1.4.48)
