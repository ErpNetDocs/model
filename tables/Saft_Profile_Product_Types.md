# Table Saft_Profile_Product_Types


## Entity

Entity: [Regulatory.Saft.ProfileProductTypes](~/entities/Regulatory.Saft.ProfileProductTypes.md)

A mapping table defining the relationship between a product type and a SAF-T ProductType code within a SAFT profile. It stores which product type corresponds to which SAF-T ProductType code for the selected SAFT profile when generating the SAF-T file. Entity: Saft_Profile_Product_Types (Introduced in version 27.1.0.76)
