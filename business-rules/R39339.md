# R39339 CustomerExternalAccess – Serviced By Enterprise Company Location Required
+-----------------------------+---------------------------------------------------------------------------------------+
| **Code**                    | R39339                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | @Crm.Sales.CustomerExternalAccess                                                     |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | ServicedByEnterpriseCompanyLocationRequired                                           |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | Customer, Customer.ServicedByEnterpriseCompanyLocation                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | Commit                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Record State**            | Inserted, Updated                                                                     |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | The rule validates that External Access can be created or updated only for Customers  | 
|                             | that have a Serviced By Enterprise Company Location defined.                          |  
+-----------------------------+---------------------------------------------------------------------------------------+
| **Description**             | The rule prevents creating External Access records that cannot be accessed in the     | 
|                             | Client Center. The rule prevents creating External Access records that cannot be      |
|                             | accessed in the Client Center. It ensures that External Access can be created only    |
|                             | for customers who have a Serviced By Enterprise Company Location defined.             |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Message**                 | External access cannot be activated for customer “{Customer.DisplayText}”.            | 
|                             | The Customer does not have a defined Serviced by Enterprise Company Location.         |  
|                             | <br><br>                                                                              |    
|                             | Please set a Serviced by Enterprise Company Location in the definition of             | 
|                             | “{Customer.DisplayText}” before creating an External Access record.                   |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Version**                 | Introduced: v.                                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Revocable**               | Yes                                                                                   |
+-----------------------------+---------------------------------------------------------------------------------------+
