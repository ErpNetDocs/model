# R39281 Offers - Set Customer From To Party
+------------------+------------------------------------------------------------------------------------------------+
| **Code**         | R39281                                                                                         |
+------------------+------------------------------------------------------------------------------------------------+
| **Entity**       | @Crm.Presales.Offers                                                                           |
+------------------+------------------------------------------------------------------------------------------------+
| **Name**         | SetCustomerFromToParty                                                                         |
+------------------+------------------------------------------------------------------------------------------------+
| **Attribute**    | Customer                                                                                       |
+------------------+------------------------------------------------------------------------------------------------+
| **Layer**        | Back-End                                                                                       |
+------------------+------------------------------------------------------------------------------------------------+
| **Events**       | Releasing +                                                                                    |
+------------------+------------------------------------------------------------------------------------------------+
| **Priority**     | Normal                                                                                         |
+------------------+------------------------------------------------------------------------------------------------+
| **Modify**       | YES                                                                                            |
+------------------+------------------------------------------------------------------------------------------------+
| **Action**       | If (Customer == NULL):<br>                                                                     |
|                  | - Create a new Customer record where:<br>                                                      |
|                  |   • Party = Offer.ToParty<br>                                                                  |
|                  |   • Customer.FromDate = TODAY<br>                                                              |
|                  |   • Customer.EnterpriseCompany = Offer.EnterpriseCompany<br>                                   |
|                  |   • Customer.DefaultCurrency = Offer.EnterpriseCompany.BaseCurrency<br>                        |
|                  |   • Customer.CreditLimit = Offer.EnterpriseCompany.DefaultCustomerCreditLimitBase<br>          |
|                  |   • Other customer fields are set to NULL or to their default values<br>                       |
|                  | - Then set the new Customer to the Offer's Customer field.                                     |
+------------------+------------------------------------------------------------------------------------------------+
| **Description**  | The rule automates the creation and assignment of a Customer when an Offer is approved         |
|                  | (system state set to Released or higher). It eliminates the need for manual customer creation  |
|                  | by the user.                                                                                   |
+------------------+------------------------------------------------------------------------------------------------+
| **Message**      | \-                                                                                             |
+------------------+------------------------------------------------------------------------------------------------+
| **Version**      | Introduced:                                                                                    |
+------------------+------------------------------------------------------------------------------------------------+
| **Revocable**    | NO                                                                                             |
