# R101753 Case - Case Development Info On Key Field Changes
+-----------------------------+---------------------------------------------------------------------------------------+
| **Code**                    | R101753                                                                               |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | @Projects.Agile.Cases                                                                 |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | CaseDevelopmentInfoOnKeyFieldChanges                                                  |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | Title, Project, ProjectArea, ProjectMilestone, Priority                               |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | Commit, Committed                                                                     |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Record State**            | Inserted, Updated                                                                     |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | YES                                                                                   |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | If any of the Title, Project, Project Area, Project Milestone, or Priority fields have been changed<br>
|                             | , then create a new CaseDevelopment record for each changed field as follows:<br>
|                             | <br>
|                             | - if the Title field has been changed – for the current Case, create a new CaseDevelopment where<br>
|                             | DevelopmentType = Edit and Description = "Title set to '{Title}'" (on insert) or <br>
|                             | Description = "Title changed from '{Title.DatabaseValue}' to '{Title}'" (on update)<br>
|                             | <br>
|                             | - if the Project field has been changed – for the current Case, create a new CaseDevelopment where<br>
|                             | DevelopmentType = Edit and Description = "Project set to '{Project.Name}'" (on insert) or <br>
|                             | Description = "Project changed from '{Project.DatabaseValue.Name}' to '{Project.Name}'" (on update)<br>
|                             | <br>
|                             | - if the Project Area field has been changed – for the current Case, create a new CaseDevelopment where<br>
|                             | DevelopmentType = Edit and Description = "Project Area set to '{ProjectArea.Name}'" (on insert) or <br>
|                             | Description = "Project Area changed from '{ProjectArea.DatabaseValue.Name}' to '{ProjectArea.Name}'" (on update)<br>
|                             | <br>
|                             | - if the Project Milestone field has been changed – for the current Case, create a new CaseDevelopment where<br>
|                             | DevelopmentType = Edit and Description = "Project Milestone set to '{ProjectMilestone.Name}'" (on insert) or <br>
|                             | Description = "Project Milestone changed from '{ProjectMilestone.DatabaseValue.Name}' to '{ProjectMilestone.Name}'" (on update)<br>
|                             | <br>
|                             | - if the Priority field has been changed – for the current Case, create a new CaseDevelopment where<br>
|                             | DevelopmentType = Edit and Description = "Priority set to {Priority}" (on insert) or <br>
|                             | Description = "Priority changed from {Priority.DatabaseValue} to {Priority}" (on update)<br>
+-----------------------------+---------------------------------------------------------------------------------------+
| **Description**             | The rule ensures that changes to the Title, Project, Project Area, Project Milestone, |
|                             | and Priority fields of a case are tracked by creating a new case development of type  |
|                             | Edit for each changed field.                                                          |
