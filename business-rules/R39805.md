# R39805 WarehouseOrder - Create Warehouse Transactions

| Name | Value |
| ---- | ----- |
| Code | R39805 |
| Entity |@Logistics.Wms.WarehouseOrders |
| Name | CreateWarehouseTransactions |
| Attribute | SnapshotQunatity, SnapshotQuantityBase, CountedQuantity, CountedQuantityBase, QuantityUnit |
| Layer | Back-End |
| Events | Releasing + |
| Modify | YES |
| Action | Validates all Warehouse Reconciliation Details. <br> Processes only details with Review Status = Approved.<br> For each approved detail:<br> - calculates the difference between SnapshotQuantity and CountedQuantity <br> - calculates the difference between SnapshotQuantityBase and CountedQuantityBase <br> - determines Direction by the difference in Quantity <br> - populates Quantity from the difference in Quantity <br> - populates QuantityBase from the difference in QuantityBase <br> - populates StandardQuantity as Quantity converted according to the measurement ratio <br> - creates one Warehouse Transaction <br><br> For products with variable measurement ratios, Direction is determined by Quantity, and QuantityBase is populated with the signed base quantity difference. |
| Description | Use the Description from @serial-numbers-business-rules |
| Message | Use the Message from @serial-numbers-business-rules |
| Version | Introduced in: 26.2 |
| Revocable | YES |
