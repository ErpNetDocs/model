# R39608 Supplier - Cannot Change Party
| Name | Value |
| ---- | ----- |
| Code | R39608 |
| Entity | @Logistics.Procurement.Suppliers |

| Attribute | Party |
| Layer | Back-End |

| Priority | Normal |
