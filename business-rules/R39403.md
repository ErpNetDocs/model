# R39403 Lead - Check Enterprise Company Location
+-----------------------------+---------------------------------------------------------------------------------------+
| **Code**                    | R39403                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | Crm.Presales.Leads Entity                                                             |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | CheckEnterpriseCompanyLocation                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | EnterpriseCompanyLocation                                                             |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | Commit                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | Validate that:                                                                        |
|                             | <br></br>                                                                             |
|                             | If (EnterpriseCompany != null AND EnterpriseCompanyLocation !=null)                   |                                           
|                             | <br></br>                                                                             |
|                             | EnterpriseCompanyLocation.Company == EntepriseCompany.Company                         |
|                             | <br></br>                                                                             |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Description**             | The current validation checks that if there is a selected company location, it is a   |
|                             | location of the enterprise company of the lead. Leads with enterprise company location| 
|                             | which are not part of the enterprise company of the specified lead would be           |
|                             | inconsistency.                                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Message**                 | The selected Enterprise Company Location '{EnterpriseCompanyLocation.DisplayText}' is |                         
|                             | not a company location of the Lead’s Enterprise Company                               |
|                             | '{EnterpriseCompany.DisplayText}', which is not allowed.                              |
|                             | <br></br>                                                                             |
|                             | Select an Enterprise Company Location that belongs to the specified Enterprise        |                       
|                             | Company and try again.                                                                |                      
|                             | <br></br>                                                                             |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Version**                 | Introduced: v.                                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Revocable**               | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
