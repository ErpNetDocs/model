# R39390 Accounting Voucher - Completed State Cannot Be Revoked
+-----------------------------+---------------------------------------------------------------------------------------+
| **Code**                    | R39390                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | @Finance.Accounting.AccountingVouchers                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | CompletedStateCannotBeRevoked                                                         |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | State                                                                                 |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | State Changing                                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | If (FromState >= Completed)                                                           |
|                             | <br></br>                                                                             |
|                             | Then (ToState cannot be Released or lower than Released)                              |   
+-----------------------------+---------------------------------------------------------------------------------------+
| **Description**             | Ensures the integrity of accounting data submitted to SAF-T by preventing further state changes after completion.
+-----------------------------+---------------------------------------------------------------------------------------+
| **Message**                 | Invalid system state change.                                                          |
|                             | <br></br>                                                                             |
|                             | The state Completed of the Accounting Vouches cannot be revoked.                      |                 
+-----------------------------+---------------------------------------------------------------------------------------+
| **Version**                 | Introduced: v.26.2                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Revocable**               | YES                                                                                   |
+-----------------------------+---------------------------------------------------------------------------------------+
