# R39803 WarehouseOrder - Set Reconciliation Details Review Status to Finished

| Name | Value |
| ---- | ----- |
| Code | R39803 |
| Entity | Logistics.Wms.WarehouseOrders |
| Name | SetReconciliationDetailsReviewStatusToFinished |
| Attribute |WarehouseReconciliationDetails.ReviewStatus |
| Layer | Back-End                                        |
| Events | Completed |
| Priority | Normal |
| Modify | YES |
| Applicable Legislations | ALL // no condition needed |
| Action | Set ReviewStatus = Finished <br> in WarehouseReconciliationDetails <br> Where WarehouseReconciliationDetails.WarehouseOrder = WarehouseOrderId|
| Description |When a Warehouse Order related to a Warehouse Reconciliation is completed, the rule must automatically find all related Warehouse Reconciliation Details <br> and set their ReviewStatus to Finished. <br> The relation is determined through the WarehouseOrder field in WarehouseReconciliationDetails, <br> which stores the Warehouse Order assigned to each detail during the order creation process. <br> The rule must affect all and only those Warehouse Reconciliation Details whose WarehouseOrder is equal to the current document being completed.|
| Message |-|
| Introduced In Version | Introduced: 26.2 |
| Revocable | NO |
