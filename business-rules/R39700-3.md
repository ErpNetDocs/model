# R39700-3 User - Set Default Language

| Name | Value |
| ---- | ----- |
| Code | R39700-3 |
| Entity | @Systems.Security.Users|
| Name | SetDefaultLanguage|
| Attribute | DefaultLanguage |
| Layer | Back-End |
| Events | Commit|
| Record State| Insert |
| Priority | Normal|
| Modify | YES |
| Action | On Insert<br>If User.DefaultLanguage != NULL <br>THEN Return,<br>ЕLSE <br>(If User.DefaultLanguage == NULL <br>AND Config(where KeyPath == "Default language" && IsActive == True && KeyValue != NULL && KeyValue != "0").Count() == 1 <br> THEN set User.DefaultLanguage = Config.KeyValue<br>ELSE User.DefaultLanguage = {$language} // current Language)|
| Description| The rule sets the Default Language equal to the value of the config key ["Default Language"](https://docs.erp.net/tech/reference/config-options-reference.html#6-default-language).<br>If the key is NULL or not defined, the language is equal to the current language of the logged creation user|
| Message | /-|
| Introduced |v. 26.2 |
| Revocable | YES |
