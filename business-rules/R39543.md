# R38379 Case Development - Notify All Case Implicit Followers
+-----------------------------+---------------------------------------------------------------------------------------+
| **Code**                    | R39543                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | @Projects.Agile.CaseDevelopments                                                      |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | NotifyAllCaseImplicitFollowers                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | \-                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | Committed                                                                             |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Late                                                                                  |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | YES                                                                                   |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | The rule performs the following steps to determine which users should be notified:    |   
|                             | - Collect all non-null Case header references (e.g., Project, Project Area, Project Milestone, Sprint, Case Category, etc.)<br>    
|                             | - Build a list of referenced objects<br>                                              |
|                             | - Retrieve users from FollowedEntities who follow any of these objects with FollowLevel = Favorite.<br>
|                             | - Remove duplicate users and exclude CaseDevelopment.CreationUser<br><\br>            |
|                             | If any users are found, the rule creates a notification for each of them through the @notifications system. 
+-----------------------------+---------------------------------------------------------------------------------------+
| **Description**             | The rule notifies users who follow at Favorite level at least one entity referenced   |     
|                             | in the Case header when a Case Development is saved, so they receive timely updates   |
|                             | about creations and changes on Cases related to objects of interest.                  |
+-----------------------------+---------------------------------------------------------------------------------------+

| **Revocable**               | YES                                                                                   |
