# R39658 Accounting Voucher - Reporting Currency Amounts Equal Movement Amounts
+-----------------------------+--------------------------------------------------------------------------------+
| **Code**                    | R39658                                                                         |
+-----------------------------+--------------------------------------------------------------------------------+
| **Entity**                  | @Finance.Accounting.AccountingVouchers                                         |
+-----------------------------+--------------------------------------------------------------------------------+
| **Name**                    | ReportingCurrencyAmountsEqualMovementAmounts                                   |
+-----------------------------+--------------------------------------------------------------------------------+
| **Attribute**               | AccountingVoucherLine.Debit, AccountingVoucherLine.DebitReporting,<br>         | 
|                             | AccountingVoucherLine.Credit, AccountingVoucherLine.CreditReporting            |                     
+-----------------------------+--------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                       |
+-----------------------------+--------------------------------------------------------------------------------+
| **Events**                  | Releasing +                                                                    |
+-----------------------------+--------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                         |
+-----------------------------+--------------------------------------------------------------------------------+
| **Modify**                  | NO                                                                             |
+-----------------------------+--------------------------------------------------------------------------------+
| **Action**                  | If AccountingVoucherLine.Currency=EnterpriseCompany.ReportingCurrency<br>      |
|                             | and DocumentDate <= EnterpriseCompany.ReportingCurrencyEndDate <br>            |
|                             | OR EnterpriseCompany.ReportingCurrencyEndDate Is Null<br>                      |
|                             | for each voucher line validate that <br>                                       |
|                             | Debit = DebitReporting and Credit = CreditReporting                            |
|-----------------------------+--------------------------------------------------------------------------------+
| **Description**             | Upon releasing the voucher, if the document date is before the <br>            |    
|                             | Reporting Currency End Date and the currency of the voucher line is the<br>    |
|                             | enterprise company’s reporting currency,                                       |
|                             | each line is validated to ensure that Debit = Debit Reporting and <br>         | 
|                             | Credit = Credit Reporting.                                                     |
|-----------------------------+--------------------------------------------------------------------------------+
| **Message**                 | In line {LineNo} of document {DocumentText}, Debit amount is not equal to<br>  |
|                             | Debit Reporting amount or Credit amount is not equal to Credit Reporting <br>  |
|                             | amount. <br>                                                                   |   
|                             | When the currency of the accounting voucher line is the reporting currency,<br>|
|                             | Debit must equal Debit Reporting and Credit must equal Credit Reporting.<br>   |
|                             | _Sort Messages By Attribute: LineNo_                                           |                 
+-----------------------------+--------------------------------------------------------------------------------+
| **Version**                 | Introduced: v.26 SP9                                                           |
+-----------------------------+--------------------------------------------------------------------------------+
| **Revocable**               | YES                                                                            |
+-----------------------------+--------------------------------------------------------------------------------+
