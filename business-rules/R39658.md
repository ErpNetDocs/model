# R39658 AccountingVoucher - Debit Equals Debit Reporting And Credit Equals Credit Reporting
+----------------------+-----------------------------------------------------------------------------------------------+
| **Code**             | R39658                                                                                        |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Entity**           | @Finance.Accounting.AccountingVouchers                                                        |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Name**             | DebitEqualsDebitReportingAndCreditEqualsCreditReporting                                       |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Attribute**        | AccountingVoucherLine.Debit, AccountingVoucherLine.DebitReporting,AccountingVoucherLine.Credit, AccountingVoucherLine.CreditReporting|
+----------------------+-----------------------------------------------------------------------------------------------+
| **Layer**            | Back-End                                                                                      |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Events**           | Releasing +                                                                                   |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Priority**         | Normal                                                                                        |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Modify**           | NO                                                                                            |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Action**           | If AccountingVoucherLine.Currency = EnterpriseCompany.ReportingCurrency <br>                  |
|                      | and DocumentDate <= EnterpriseCompany.ReportingCurrencyEndDate <br>                           |
|                      | OR EnterpriseCompany.ReportingCurrencyEndDate Is Null <br>                                    |
|                      | for each voucher line validate that <br>                                                      |
|                      | Debit = DebitReporting and  Credit = CreditReporting                                          |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Description**      | Upon releasing the voucher, if the document date is before the Reporting Currency End Date<br>|
|                      | and the currency of the voucher line is the enterprise company’s reporting currency, <br>     |
|                      | each line is validated to ensure that Debit = Debit Reporting and Credit = Credit Reporting.  |                  
+----------------------+-----------------------------------------------------------------------------------------------+
| **Message**          | In line {LineNo} of document {DocumentText}, Debit amount is not equal to Debit RC amount<br> | 
|                      | or Credit amount is not equal to Credit RC amount.<br>                                        |
|                      | When the currency of the accounting voucher line is the reporting currency, Debit must equal<br>|
|                      | Debit RC and Credit must equal Credit RC.<br>                                                 |
|                      | *Sort Messages By Attribute: LineNo*                                                          |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Version**          | Introduced:                                                                                   |
+----------------------+-----------------------------------------------------------------------------------------------+
| **Revocable**        | YES                                                                                           |
+----------------------+-----------------------------------------------------------------------------------------------+
