# R39677 WarehousePolicies - Counting Order Document Type Value

| Name | Value |
| ---- | ----- |
| Code | R39677 |
| Entity | @Logistics.Wms.WarehousePolicies |
| Name | CountingOrderDocumentTypeValue |
| Attribute | Value |
| Layer | Back-End |
| Events | ClientCommit |
| Priority | Normal |
| Modify | NO |
| Applicable Legislations | ALL // no condition needed |
| Action | Validate that: <br/> WarehousePolicy.Value == WarehouseOrders.Code <br/>,  WHERE DocumentType.EntityName == WarehouseOrders |
| Description | Validates that the value of the CountingOrderDocumentType policy is the Code of a Document Type whose Entity Name is WarehouseOrders.
If the value does not correspond to such a document type, an error is thrown. |
| Message | The 'Value' field contains an invalid value. <br/> The selected value '{Value}' is invalid for a warehouse policy of kind '{PolicyKind}'. <br/> Valid values for warehouse policies of this kind are only Warehouse Orders document type Codes. |
| Introduced In Version | Introduced: 24  |
| Revocable | NO |
