# R39538 SocialComment - Reply Must Reference Same Data Object
+-----------------------------+---------------------------------------------------------------------------------------+
| **Code**                    | R39538                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | @Communities.Social.Comments                                                          |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | ReplyMustReferenceSameDataObject                                                      |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | ReplyToComment                                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | Commit                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | On commit, validate `ReplyToComment` references a comment from the same Data Object.  |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Description**             | Prevents creation of reply comments that reference parent comments from a different   |
|                             | data object.                                                                          |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Message**                 | Reply must reference a comment on the same data object.                               |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Version**                 | Introduced: v.26.2                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Revocable**               | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
