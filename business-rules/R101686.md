# R101686 Document - Check User Status Consistency
+-----------------------------+---------------------------------------------------------------------------------------+
| **Code**                    | R101686                                                                               |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | @General.Documents.Documents                                                          |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | CheckUserStatusConsistency                                                            |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | UserStatus                                                                            |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | COMMIT                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | If Documents.UserStatus != null, then                                                 |
|                             | DocumentTypeUserStatuses.State == Documents.State                                     |
|                             | AND DocumentTypeUserStatuses.Active == true                                           |
|                             | AND DocumentTypeUserStatuses.DocumentType ==                                          |
|                             | Documents.DocumentType                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Description**             | The rule ensures that a document cannot be saved with a user status that              |
|                             | is not valid for the current system state and document type.                          |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Message**                 | Saving {Document.DisplayText} is not allowed because the selected user                |
|                             | status is not valid for the current system state of the document.                     |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Version**                 | Introduced: v....                                                                     |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Revocable**               | Yes                                                                                   |
+-----------------------------+---------------------------------------------------------------------------------------+
