# R39576 WarehouseReconciliations – Warehouse Zone Is Always Null

| Name | Value |
| ---- | ----- |
| Code | R39576 |
| Entity | @Logistics.Wms.WarehouseReconciliations |
| Name | WarehouseZoneIsAlwaysNull |
| Attribute | WarehouseZone |
| Layer | Back-End |
| Events | Commit |
| Priority | Early |
| Modify | NO |
| Applicable Legislations | ALL // no condition needed |
| Action | Validate that: <br/> WarehouseZone == Null |
| Description | The rule validates that the Warehouse Zone field is always empty. For reconciliation purposes, Warehouse Zones are selected in the Warehouse Reconciliation Lines. |
| Message | Warehouse Zone in the document header is no longer used and must remain empty. <br> For reconciliation purposes, Warehouse Zones must be specified in the Warehouse Reconciliation Lines. <br> Please select the Warehouse Zone in the lines and try again. |

| Revocable | NO |
