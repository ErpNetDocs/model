# R37555 Case - Closed Only If Resolved

| **Code**                    | R39411                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | @Projects.Agile.Cases                                                                 |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | ClosedOnlyIfTaskCompleted                                                             |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | SystemState                                                                           |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | Commit                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | If System State is changed to 'CLOSED'                                                |
|                             | <br></br>                                                                             |

|                             | Please complete all related Tasks before closing the Case.                            |
