# R39596 Checklist - Create Checklist Execution
+-----------------------------+---------------------------------------------------------------------------------------+
| **Code**                    | R39596                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Entity**                  | @Applications.Service.ServiceActivities                                               |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Name**                    | CreateChecklistExecution                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Attribute**               | Document State                                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Layer**                   | Back-End                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Events**                  | FirmPlanning +                                                                        |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Priority**                | Normal                                                                                |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Modify**                  | NO                                                                                    |
+-----------------------------+---------------------------------------------------------------------------------------+
| **Action**                  | If (State == FirmPlanned)<br></br>                                                    |
|                             | For all existing line records search Service and check where:<br>                     |
|                             | (Service.OperationalProcedure != null )<br>                                           |
|                             | For each found Operational Procedure check if there is OperationalProcedureExecution; |
|                             | if missing, create it.<br>                                                            |
|                             | For each checklist template assigned to the operational procedure, create the<br>     |
|                             | corresponding Checklist Execution Items by copying all Checklist Template Items<br>   |
|                             | into the execution (one execution item per template item).<br></br>                   |
|                             | The rule must be idempotent (must not create duplicates when triggered more than once)|
+-----------------------------+---------------------------------------------------------------------------------------+
| **Description**             | This rule automatically initializes operational work execution and checklists when a  |
|                             | Service Activity becomes Firm Planned.                                                |
|                             | For all services included in the activity that have a defined Operational Procedure,  |
|                             | the system creates the required Operational Procedure Executions and populates the    |
|                             | related Checklist Execution Items from the procedure’s checklist templates. This      |
|                             | ensures that technicians receive a ready-to-use checklist at the planning stage and   |
|                             | eliminates manual checklist setup. Services without an operational procedure are      |
|                             | ignored.                                                                              |
+-----------------------------+---------------------------------------------------------------------------------------+
